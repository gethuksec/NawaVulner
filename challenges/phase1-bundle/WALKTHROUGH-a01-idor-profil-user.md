# Walkthrough uji: `a01-idor-profil-user` (IDOR profil pemohon)

## Ringkas

1. Login platform, buka `/challenges/a01-idor-profil-user`, **Buka lab**.
2. Dari beranda, buka tautan ringkasan permohonan (URL berisi `profile?id=`).
3. Enumerasi nomor referensi di URL (dan opsional `GET .../api/profile?id=`) sampai muncul blok **catatan verifikator** berisi flag.
4. Salin flag ke submit di platform.
